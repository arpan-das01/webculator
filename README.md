# WebCulator

A simple web-based calculator built with HTML, CSS & JavaScript.

🔗 [Live Demo](https://arpan-das01.github.io/webculator/)

![Screenshot](screenshot.png)

## Features

Supports the four basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

## Usage

Clone the repo and open `index.html` in any browser — no build step required.

## Future Improvements

- Keyboard input support
- Additional buttons
- Replace `eval()` with a stack-based `calculate()` function
